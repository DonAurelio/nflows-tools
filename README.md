# nFlows: Tools

Command-line utilities for validating nFlows output and analyzing execution results

### Requirements

1. **OS:** Ubuntu 22.04 
2. **Software:** `python3.10`, `graphviz-dev`, `libgraphviz-dev` o similar.

### Install

```sh
pip install .
```

### Uninstall

```sh
pip uninstall nflows_tools
```
